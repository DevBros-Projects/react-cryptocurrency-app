# React-cryptocurrency-app

First project by DEVBROS.

## About

This is a single page web app built with React and Tailwind CSS that displays the current prices of various cryptos.

## Project Details

### Packages Installed

- React icons `npm i react-icons`
- Dotenv `npm i dotenv `
- Axios ` npm i axios`

### Custom Fonts

- Poppins, Sans-serif
- Karla, Sans-serif
